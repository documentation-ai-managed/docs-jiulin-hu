<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Launched core dashboard at `https://dashboard.example.com`
- Implemented OAuth2 authentication flow via `https://auth.example.com`

## Breaking Changes

- Updated API base URL from `/v0` to `/v1`
- Removed legacy `{api_key}` header; use `Authorization: Bearer {token}` instead

</Update>

## Upgrading to v1.2.0

Follow these steps to upgrade smoothly. Test in a staging environment first.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export current configurations and data exports.

    <Callout kind="tip">
      Use the new export API: `POST https://api.example.com/v1/exports`
    </Callout>
  </Step>
  <Step title="Update SDK" icon="package">
    Install the latest version across platforms.
    
    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install jiulin-hu@1.2.0
    ````
    ````bash
    yarn add jiulin-hu@1.2.0
    ````
    </CodeGroup>
  </Step>
  <Step title="Migrate Auth" icon="shield">
    Update your integration to use Bearer tokens.

    <CodeGroup tabs="JavaScript,Python">
    ````javascript
    const response = await fetch('https://api.example.com/v1/users', {
      headers: {
        'Authorization': `Bearer ${YOUR_TOKEN}`
      }
    });
    ````
    ````python
    import requests
    response = requests.get(
      'https://api.example.com/v1/users',
      headers={'Authorization': f'Bearer {YOUR_TOKEN}'}
    )
    ````
    </CodeGroup>
  </Step>
  <Step title="Test & Deploy" icon="rocket">
    Run integration tests and monitor for `>99.9%` uptime.
  </Step>
</Steps>

<Callout kind="alert">
  **Deprecation Notice:** The old `/v0` endpoints will be removed in v1.3.0. Migrate now to avoid disruptions.
</Callout>
