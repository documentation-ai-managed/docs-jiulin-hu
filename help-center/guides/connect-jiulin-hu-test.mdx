## Overview

Follow these steps to connect to the Jiulin Hu test API.

<Steps>
  <Step title="Get Your API Key">
    Visit `https://jiulinhuproduct.com/api` and generate an API key.
  </Step>
  <Step title="Make Your First Request">
    ````bash
    curl -H "Authorization: Bearer YOUR_API_KEY" \
      https://api.jiulinhuproduct.com/v1/ping
    ````
  </Step>
</Steps>
