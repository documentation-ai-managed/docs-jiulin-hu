Whether you build web apps, APIs, or mobile services, Jiulin Hu reduces debugging time and boosts reliability.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Install" icon="download">
    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm install -g jiulin-hu
      ```
      ```bash
      yarn global add jiulin-hu
      ```
      ```bash
      pnpm add -g jiulin-hu
      ```
    </CodeGroup>
  </Step>
  <Step title="Initialize" icon="settings">
    Create a config file in your project root.

````json
{
  "testDir": "tests/",
  "reporter": "html",
  "parallel": true
}
````
  </Step>
  <Step title="Run Tests" icon="play">
    Execute your first test suite.

    ```bash
    jiulin-hu run
    ```
  </Step>
</Steps>
